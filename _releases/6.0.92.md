# Security Fixes
* Prevent command injection in *rpmuncompress*(1) (CVE-2026-44604)
* Harden the ndb implementation against integer overflows (CVE-2026-44605)
* Fix possible crashes in case of memory allocation failures (#4233)

# Building Packages
* Fix regression in RubyGems extraction causing double dot in `.gemspec` suffix
  (RhBug:2492128)
* Fix spurious whitespace warning when using `%define` (RhBug:2492483) (#4249)
* Fix broken `%add_sysuser m` directive (#4232)

# Signing & Verifying Packages
* Fix *rpmsign*(1) to handle `%_file_signing_key_id` values greater than
  `INT_MAX` (#4218)
